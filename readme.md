# 索雷伯PM100D功率计控制软件使用手册

## 1. 软件概述
PM100D功率计控制软件是一款用于连接和控制PM100D系列光功率计的图形化界面工具，支持实时数据显示、参数设置和数据记录功能。

## 2. 系统要求
- 操作系统：Windows 7/10/11
- 驱动要求：NI-VISA
- 硬件要求：PM100D功率计设备及连接线缆

## 3. 软件安装
1. 下载软件压缩包并解压
2. 双击运行`PM100D功率计.exe`（无需额外安装）

## 4. 界面说明

### 4.1 主界面布局

(左侧区域) | (中间分隔线) | (右侧区域)
----------|--------------|-----------
串口设置区 |              | 数据显示区

### 4.2 功能区域详解

#### 串口设置区（左侧）
- **设备选择**：通过下拉菜单选择连接的设备
- **检查串口**：点击扫描可用串口设备
- **串口信息**：显示当前连接的串口详细信息

#### 数据显示区（右侧）
- **通道控制**：
  - 通道自定义名称：支持自定义输入通道名称，默认为通道1  
  - 波长设置：选择预设波长或输入自定义值
  - 补偿设置：输入自定义补偿值
  - 功率单位：自动显示当前设备功率单位（W/dBm）
  - 实时数值：7位LCD显示当前功率值
  - 极值记录：显示测量过程中的最大值和最小值
- **图形显示区**：功率变化曲线图（需连接后显示）
- **控制按钮区**：6个功能按钮

## 5. 操作指南

### 5.1 设备连接
1. 确保功率计通过USB/串口与电脑连接
2. 点击"检查串口"按钮扫描可用端口（连接前会自动刷新可用设备列表）
3. 从下拉菜单中选择正确的COM端口，通常为类似USB0::4883::32888::P0047129::0::INSTR,其中P0047129为设备序列号
4. 点击"链接功率计"建立连接

### 5.2 基本测量
1. 成功连接后，LCD显示屏将显示实时功率值
2. 波长设置：
   - 从下拉菜单选择预设波长（850nm/1310nm等）
   - 或直接输入自定义波长值
   - 点击"设置"按钮应用新波长

### 5.3 数据记录
1. 点击"开始记录"开始保存测量数据
2. 实时数据将显示在图形区域
3. 点击"停止记录"结束数据采集
4. 极值（最大/最小值）将自动更新显示

### 5.4 其他功能
- **自动量程**：
  - "开启自动范围"：允许设备自动调整测量范围
  - "关闭自动范围"：固定当前量程
- **清空数据**：点击"清空"重置所有显示数据
- **断开连接**：点击"断开功率计"结束会话（后台线程安全断开，避免界面卡顿）
- **自动化接口**：软件内置 TCP 自动化服务器，支持外部程序远程控制连接、读数和参数设置

## 6. 注意事项
1. 首次使用前请确保已安装正确的驱动程序
2. 连接失败时请检查：
   - 设备电源是否开启
   - 线缆连接是否牢固
   - COM端口选择是否正确
3. 测量高功率信号时请注意不要超过设备量程
4. 长时间不使用时请断开设备连接

## 7. 常见问题解答

Q: 软件无法检测到设备怎么办？
A: 1) 检查设备电源 2) 更换USB端口 3) 重启软件 4) 检查驱动

Q: 测量数值不稳定如何解决？
A: 1) 检查光纤连接 2) 关闭附近干扰源 3) 尝试固定量程模式

Q: 数据记录保存在哪里？
A: 数据默认保存在程序所在目录的Record文件夹下

Q: 断开连接后重新连接失败怎么办?
A: 属于设备驱动问题，重启软件并重新插拔 USB 线缆后再次尝试连接

Q: 如何查看版本更新记录?
A: 点击界面上的版本号标签，可弹出"关于"窗口查看完整更新历史
